This is the react-native client.

Application is developed to using Expo and so expo cli and mobile-client are needed to run the app.

Instructions to run:
1. Install dependencies: npm install
2. Install expo-clie: npm install -g expo-cli
3. Install expo mobile client for Android from Play Store.
4. Specify the BACKEND_DOMAIN in config.js in this folder. 
5. Ensure that the backend is running and connected to same private network.
6. Run 'expo start' in this folder.
7. Scan the displayed QR-code using the expo mobile client.
